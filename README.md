# Нейро-станция «МНИСТ-1»

Интерактивный пульт в стиле советской лаборатории: настоящая нейросеть 784→30→10,
написанная с нуля (init · forward · backward · SGD), обучается на реальном MNIST
прямо в браузере. Никаких фреймворков и сервера — только JS.

**Запуск:** https://mikky-a.github.io/mnist-station/

Основано на книге Майкла Нильсена «Neural Networks and Deep Learning»
(http://neuralnetworksanddeeplearning.com), глава 1.

Данные: MNIST (50 000 обучающих + 10 000 контрольных), НИСТ США, 1998.
